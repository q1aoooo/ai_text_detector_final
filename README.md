# AI-Generated Text Detection

This project explores and compares three different machine learning models for the task of detecting AI-generated text. The goal is to build a robust classifier that can distinguish between text written by a human and text generated by a large language model.

The project was completed in Jupyter Notebooks, one for each of the models, kept in the models folder. The dataset used can be found in the dataset folder. We included it since this dataset is already preprocessed and reflects the combined version of two datasets, with only necessary columns used.

The checkpoints for best configurations found for BiLSTM and DistilBert can also be found in the checkpoints folder.
